# Typr

A keyboard app to improve typing speed.

## Features

- A box in which to type keys
- See the keys you should type next
- Detailed Metrics
  - Number of keys pressed in 5 min window of practice
  - Accuracy %

## Technical requirements

- CSS styling
- images
- Redux
- Only use the keys "asdfjkl;"

# Sections

- Type
- Data
- Chill

- Settings
